# 技能工作區使用指南

## 🎯 什麼是技能工作區？

技能工作區是一種特殊的工作區，可以：
- 📝 定義一個可重用的任務（例如：計算帳單、分析日誌）
- ⚡ 設定快捷按鈕，一鍵執行常用命令
- 🤖 讓 AI 自動理解這個技能的功能和使用方式

---

## 📖 快速開始：創建你的第一個技能

### 步驟 1：創建工作區
1. 點擊左側邊欄的 **「+ 新增工作區」** 按鈕
2. 選擇一個資料夾（例如：`C:\MySkills\CustomerBilling`）

### 步驟 2：標記為技能
1. 在左側邊欄找到剛創建的工作區
2. **右鍵點擊** 工作區名稱
3. 選擇 **「⚙ 配置」**
4. 勾選 **「📚 這是一個技能工作區」**

### 步驟 3：設定技能資訊
在配置對話框中填寫：

**技能簡述**（選填）
```
計算客戶的月度帳單，包含折扣和稅額
```

**標籤**（選填，用逗號分隔）
```
財務, 帳單, 自動化
```

**初始化命令**（選填）
```powershell
npm install
```
> 開啟工作區時自動執行

### 步驟 4：創建 skill.md
1. 在配置對話框中點擊 **「✨ 創建模板」**
2. 系統會自動開啟 `skill.md` 檔案
3. 編輯內容，描述這個技能：

```markdown
# 客戶帳單計算器

## 功能描述
自動計算客戶的月度帳單，包含：
- 基本費用
- 使用量計費
- 折扣計算
- 稅額計算

## 使用方式
1. 準備客戶資料（customer-data.csv）
2. 執行計算腳本
3. 生成 PDF 帳單

## 輸入參數
- 月份：YYYY-MM 格式
- 客戶ID：例如 CUST-001

## Agent 提示詞
You are a billing calculator assistant. Help the user calculate monthly bills for customers based on usage data and pricing rules.
```

### 步驟 5：新增快捷操作（選填）
在配置對話框的「快捷操作」區域：

1. 點擊 **「＋ 新增」**
2. 填寫：
   - **圖標**：`▶️` 或 `💰`
   - **按鈕文字**：`計算帳單`
   - **命令**：`python calculate.py --month 2026-01`
3. 點擊 **「確定」**

可以新增多個快捷按鈕，例如：
- `📧 發送郵件` → `python send_invoice.py`
- `📊 生成報表` → `python generate_report.py`

### 步驟 6：儲存配置
點擊對話框右下角的 **「儲存」** 按鈕

## 🚀 使用技能

### 方法 1：從技能庫開啟
1. 點擊左側邊欄底部的 **「📚 技能庫」** 按鈕
2. 在技能庫中找到你的技能
3. 點擊卡片開啟工作區

### 方法 2：直接點擊快捷按鈕
在技能庫中，每個技能卡片下方會顯示你設定的快捷按鈕：
- 點擊 **「▶️ 計算帳單」** → 自動切換到該工作區並執行命令

### 方法 3：使用 Copilot AI
1. 開啟技能工作區
2. 點擊左側的 **「💬 Copilot」** 圖標
3. 輸入問題，例如：
   ```
   如何計算客戶 A 的帳單？
   ```
4. AI 會自動讀取 `skill.md`，理解這個技能的功能，並給出精準的回答

## 🔍 管理技能

### 搜尋技能
在技能庫的搜尋框輸入關鍵字：
- 搜尋名稱：`帳單`
- 搜尋標籤：`財務`
- 搜尋描述：`計算`

### 標籤篩選
點擊技能庫上方的標籤按鈕：
- `財務` → 只顯示財務相關技能
- `自動化` → 只顯示自動化技能

### 編輯技能
1. 在技能卡片右上角點擊 **「⚙」** 按鈕
2. 修改配置
3. 點擊「儲存」

### 開啟資料夾
在技能卡片右上角點擊 **「📁」** 按鈕，直接開啟技能所在的資料夾

## 💡 實用範例

### 範例 1：日誌分析技能
```markdown
# 日誌分析器

## 功能
分析應用程式日誌，找出錯誤和異常

## 快捷操作
- 📊 分析今日日誌：`python analyze.py --date today`
- 🔍 搜尋錯誤：`grep -i error logs/*.log`
- 📈 生成統計：`python stats.py`
```

### 範例 2：資料庫備份技能
```markdown
# 資料庫備份助手

## 功能
自動備份 PostgreSQL 資料庫

## 快捷操作
- 💾 立即備份：`pg_dump mydb > backup.sql`
- 📂 列出備份：`ls -lh backups/`
- ♻️ 清理舊備份：`python cleanup_old_backups.py`
```

### 範例 3：客戶報表技能
```markdown
# 客戶報表生成器

## 功能
為客戶生成月度報表

## 快捷操作
- 📊 生成報表：`python generate_report.py`
- 📧 發送郵件：`python send_report.py`
- 🖨️ 列印 PDF：`python print_pdf.py`
```

## ❓ 常見問題

### Q: skill.md 一定要創建嗎？
**A:** 不一定，但強烈建議創建。有 skill.md 的好處：
- AI 能理解技能功能，給出更準確的回答
- 團隊成員能快速了解如何使用
- 作為技能的說明文件

### Q: 快捷按鈕可以執行複雜命令嗎？
**A:** 可以！例如：
```bash
python script.py --arg1 value1 --arg2 value2 && echo "完成"
```

### Q: 如何分享技能給同事？
**A:** 直接分享技能資料夾即可：
1. 將整個資料夾壓縮（包含 skill.md）
2. 同事解壓縮後，新增工作區並指向該資料夾
3. 標記為技能工作區

### Q: 可以在技能中使用環境變數嗎？
**A:** 可以！
1. 右鍵點擊工作區
2. 選擇「環境變數」
3. 新增專屬的環境變數

## 🎨 進階技巧

### 技巧 1：使用相對路徑
在 skill.md 中使用相對路徑引用檔案：
```markdown
## 設定檔
參考 `config/settings.json` 設定
```

### 技巧 2：組合多個快捷按鈕
創建一個完整的工作流程：
1. `🔄 更新資料` → `git pull`
2. `📦 安裝套件` → `npm install`
3. `🚀 啟動服務` → `npm start`

### 技巧 3：使用初始化命令
設定初始化命令自動準備環境：
```powershell
git pull && npm install && echo "環境已就緒"
```
